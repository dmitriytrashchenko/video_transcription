# video_transcription — Транскрибер видео с разделением спикеров

Desktop-приложение (tkinter) для автоматической транскрибации видео в Word-документ с разбивкой текста по говорящим и временными метками.

## Статус
Живой (в разработке) — см. [[Changelog]] и [[Known_Issues]].

## Возможности
- Извлечение аудио из видео через FFmpeg
- Разделение спикеров (диаризация) через `pyannote.audio`
- Транскрибация через OpenAI Whisper (модели `tiny`/`base`/`small`/`medium`/`large`)
- Экспорт в `.docx` с таймкодами и подписью говорящего
- Простой GUI: выбор файла, папки сохранения, языка, модели, прогресс-бар, журнал выполнения

## Установка

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**FFmpeg** нужно поставить отдельно и добавить в PATH (не входит в репозиторий — раньше полная сборка FFmpeg на 450+ МБ была случайно закоммичена в git, теперь `.gitignore`):
- Скачать с https://www.gyan.dev/ffmpeg/builds/ или через `winget install ffmpeg`

**Hugging Face токен** нужен для диаризации спикеров (`pyannote/speaker-diarization-3.1` — gated-модель):
1. Прими условия доступа на https://huggingface.co/pyannote/speaker-diarization-3.1
2. Скопируй `.env.example` в `.env` и впиши свой токен:
   ```
   HUGGINGFACE_TOKEN=hf_...
   ```

Без токена программа всё равно работает — просто без разделения спикеров (весь текст пойдёт как "Человек 1").

## Запуск
```bash
python video_transcriber.py
```

## Планы по развитию (бэклог)
Со слов автора:
1. Интерфейс на 2K-мониторе выглядит "мыльным" — нужна поддержка высокого DPI
2. Более "soft"-дизайн интерфейса вместо стандартного tkinter-вида
3. Кнопка отмены/паузы обработки — сейчас процесс нельзя прервать после запуска
4. Более подробный журнал выполнения с прогресс-барами по каждому шагу
