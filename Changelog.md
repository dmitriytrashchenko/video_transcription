# Changelog

## 2026-07-19 — Возрождение проекта: устранение утечки токена и чистка репозитория
Проект был взят из состояния "не начато" в штабе Core_operation_all_project. Найдена серьёзная проблема: реальный Hugging Face токен был захардкожен в `video_transcriber.py` и присутствовал в 3 локальных коммитах, которые ещё не были запушены на GitHub — то есть публичной утечки не произошло, но она была на подходе.

**Сделано:**
1. **Токен убран из кода** — теперь читается из `.env` (`python-dotenv`), сам токен сохранён (не отозван — решение владельца, он не был публично скомпрометирован).
2. **Локальная git-история переписана** до публикации: `git reset --soft origin/main` откатил 3 непушнутых коммита с токеном, файлы пересозданы и закоммичены заново уже без секрета в истории — токен никогда не попадёт в публичный репозиторий.
3. **`bible/` (полная сборка FFmpeg, ~453 МБ, 44 файла) убрана из git** — добавлена в `.gitignore`. FFmpeg должен ставиться отдельно, не таскаться в репозитории.
4. **README.md был не документацией, а вставленным кодом** (пользователь случайно вставил туда весь `video_transcriber.py` вместо описания проекта) — переписан на нормальную документацию. Пожелания пользователя по доработке (из конца испорченного README) сохранены в разделе "Планы по развитию".
5. Добавлен `requirements.txt` (не существовал, хотя зависимостей много: torch, whisper, pyannote.audio, librosa).
6. Добавлен `.gitignore` (.env, env/, bible/, __pycache__, *.docx).

**Не сделано в этой сессии:** живой запуск GUI (требует скачивания тяжёлых ML-моделей Whisper/pyannote — не быстрая проверка), реализация пожеланий пользователя по UI (DPI, soft-дизайн, кнопка отмены, подробный лог).
