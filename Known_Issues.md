# Known Issues / Backlog

## Не проверено живым запуском
GUI не запускался живьём в сессии 2026-07-19 — требует установки тяжёлых ML-зависимостей (torch, whisper, pyannote.audio) и скачивания моделей (сотни МБ — единицы ГБ). Требует отдельной сессии с реальным видеофайлом для проверки.

## Пожелания пользователя по UI (не реализовано)
1. Интерфейс "мыльный" на 2K-мониторе — нужна поддержка высокого DPI (Windows DPI awareness, `ctypes.windll.shcore.SetProcessDpiAwareness`)
2. Более "soft"/приятный дизайн вместо стандартного tkinter — возможно, стоит рассмотреть `ttkbootstrap` или другую тему
3. Кнопка отмены/паузы обработки — сейчас `process_video()` выполняется в потоке без возможности прервать
4. Более подробный журнал с прогресс-барами по каждому шагу, а не только общий прогресс-бар

## Диаризация требует ручного принятия условий на Hugging Face
`pyannote/speaker-diarization-3.1` — gated-модель, для работы токен должен принадлежать аккаунту, принявшему условия доступа на странице модели. Без этого `Pipeline.from_pretrained` упадёт даже с валидным токеном.

## venv в проекте битый
Старый `env/` указывает на несуществующий путь Miniconda (та же проблема, что была в других легаси-проектах) — при реальном запуске нужно пересоздать через `python -m venv venv`.
